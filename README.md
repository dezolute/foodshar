# ФудШер - Сайт для фудшеринга

Многостраничный сайт на HTML/CSS для проекта фудшеринга в Минске.

## Структура проекта

```
foodshar-website/
├── index.html          # Главная страница
├── about.html          # О нас (история, команда)
├── volunteers.html     # Информация для волонтеров
├── organizations.html  # Информация для организаций
├── contacts.html       # Контакты и форма связи
├── css/
│   └── style.css      # Все стили
└── assets/            # Папка для изображений (при необходимости)
```

## Страницы

1. **Главная (index.html)** - Hero секция, что такое фудшеринг, призывы к действию
2. **О нас (about.html)** - История движения, команда проекта
3. **Волонтерам (volunteers.html)** - Требования, как стать участником, правила
4. **Организациям (organizations.html)** - Преимущества сотрудничества, как это работает
5. **Контакты (contacts.html)** - Контактная информация и форма обратной связи

## Особенности

- ✅ Адаптивный дизайн (десктоп/планшет/мобильный)
- ✅ Современная типографика и цветовая схема (зеленая/эко-тематика)
- ✅ Единая навигация на всех страницах
- ✅ Hover-эффекты и плавные переходы
- ✅ Оптимизированные карточки и секции
- ✅ Готовая структура под контент из макета

## Как использовать

1. Откройте `index.html` в браузере
2. Навигация между страницами работает через меню
3. Все стили находятся в `css/style.css`
4. Для добавления изображений используйте папку `assets/`

## Дальнейшая доработка

- Добавить реальные изображения в папку `assets/`
- Настроить отправку формы контактов (на email или через сервис)
- Добавить карту для визуализации точек вывоза (если требуется)
- Подключить Google Analytics или Yandex.Metrika
